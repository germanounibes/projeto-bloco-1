# 🪪 Card.gen — Gerador de Cartões de Visita

> Crie, personalize e baixe cartões de visita profissionais direto no navegador. Sem frameworks, sem dependências.

---

## 🎯 Sobre o projeto

O **Card.gen** é uma aplicação web onde o usuário preenche seus dados e vê o cartão de visita sendo montado em tempo real. Ao final, pode baixar o cartão como imagem PNG.

Projeto desenvolvido como parte do **Bloco 01 — Base Sólida** do plano de estudos de front-end, cobrindo HTML semântico, CSS avançado, JavaScript vanilla e versionamento com Git.

---

## ✨ Funcionalidades

- [ ] Formulário com nome, cargo, empresa, e-mail, telefone e LinkedIn
- [ ] Preview ao vivo do cartão enquanto o usuário digita
- [ ] Seletor de paleta de cores / tema do cartão
- [ ] Seletor de layout (horizontal / vertical)
- [ ] Download do cartão como imagem PNG (via Canvas API)
- [ ] Modo claro / escuro na interface

---

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (custom properties, Grid, Flexbox, animações)
- JavaScript vanilla (DOM, eventos, Canvas API)
- Git + GitHub Pages (deploy)
